# AI Travel Planner

## Tech Stack

- Next.js
- Tailwind CSS
- TypeScript
- Express
- MongoDB
- Gemini 2.5 Flash

## Features

- Authentication
- AI Trip Planning
- Budget Estimation
- Hotel Suggestions
- Packing Assistant
- Trip Management

## Run Frontend

```bash
npm install
npm run dev
```

## Run Backend

```bash
npm install
npm run dev
```

Frontend:
http://localhost:3000

Backend:
http://localhost:5000
```
